# Threat Scanner SaaS

A defensive web security scanner built with FastAPI.

## Features

- JWT Authentication
- Async Website Scanning
- SSL / TLS Analysis
- DNS Security Checks
- Security Headers Analysis
- WAF Detection
- HTTP Methods Detection
- Open Port Checks
- robots.txt Analysis
- security.txt Analysis
- Subdomain Discovery
- CVE Lookup
- Nikto-like Checks
- Vulnerability Checks
- Scan History
- JSON Report Export
- PDF Report Export
- Real-time Scan Progress
